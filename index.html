How to use it with Claude

Claude Desktop
  1. Open Claude Desktop and go to Settings > MCP Servers.
  2. Click "Add server" and choose "Remote".
  3. Paste your generated URL in the server URL field.
  4. Name it "DeepResearch MCP" and save.
Claude.ai
  1. Open Claude.ai and go to Settings > MCP Servers.
  2. Choose "Add server" and select "Remote".
  3. Paste your generated URL, then save.
